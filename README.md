# Center of Gravity Finder is a Python project that analyzes synthetic transaction data to identify potentially suspicious accounts using network analysis, risk scoring, network relationships, temporal analysis, and geographic indicators.

## Features

- Generates Synthetic financial transaction data
- Scores accounts based on transaction behavior
- Detects multi-hop transaction networks
- Performs geographic risk analysis
- Exports results to CSV files

## Files

- generator.py - Generates synthetic transaction data
- analyzer.py - Analyzes transactions and calculates risk scores
- geo_analyzer.py - Performs geographic risk analysis
- cog_engine.py - Runs the project 
- transactions.csv - Sample transaction data

## Data Sources and Inspiration

The synthetic data model and detection logic were inspired by publicly available guidance and reporting including:

- FinCEN advisories and trend analysis
- Bank Secrecy Act (BSA) reporting Guidance
- Public reports from the DHS, DEA, FBI, and other U.S. Government Agencies
- Open source reporting on money laudering, sanctions evasion, and illicit financial networks

No real financial records or personally identifiable information are included. All transaction data used by this project is sythetically generated for educational and research purposes.
